# 🛒 Online Retail SQL Project

## 📌 Overview
This project simulates an e-commerce retail database and analyzes customer behavior, product sales, and order trends.

## 🧱 Concepts Used
- DDL (CREATE TABLE)
- DML (INSERT)
- DQL (SELECT)
- JOINS (INNER, LEFT)
- SET OPERATIONS (UNION)
- Aggregations & Business Insights
- CASE WHEN

## 📊 Key Insights
- Top customers by sales
- Most sold products
- Order trends

## 🛠 Tools
- MySQL / PostgreSQL

## 🚀 How to Run
1. Run schema.sql
2. Run data.sql
3. Run queries.sql
